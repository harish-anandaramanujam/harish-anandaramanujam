## About

I am a Data Platform Engineer with 6+ years of experience designing and operating distributed systems that turn siloed data into dependable, real-time products. I work across streaming architecture, workflow orchestration, platform reliability, developer experience, governance, and cloud cost optimization.

My core focus is Apache Airflow, Apache Flink, Apache Kafka, and modern data stores. I also build production AI/ML infrastructure and the reliable pipelines that support it.

> Platforms should make the correct path the easiest path.

## Selected impact

- Designed a Kubernetes- and Flink-based streaming architecture and custom control plane for Kafka and Kubernetes pipeline failures, reducing mean time to recovery from **8 hours to 5 minutes**. The architecture became the company-wide standard for streaming data.
- Developed **100+ high-volume, real-time pipelines**, replacing critical 24-hour batch latency with per-minute processing and enabling **25+ cross-functional teams** to process billions of records per minute.
- Built and scaled production Airflow platforms on Kubernetes, tuning schedulers, executors, autoscaling, parallelism, and resource utilization for greater throughput and reliability.
- Established CI/CD workflows with GitHub Actions and AWS CloudFormation across development, staging, and production, improving the delivery of scalable cloud infrastructure.
- Optimized large-scale Airflow ETL workloads and securely integrated Kafka, BigQuery, Redshift, and Snowflake for fault-tolerant ingestion and dependable downstream ML data.
- Led high-stakes incident response as an xMatters subject-matter expert, strengthening operational standards and minimizing service downtime.
